# Installation

## Requirements

- Go v1.10 or above

## Build from Source

```bash
git clone https://github.com/OppressionBreedsResistance/gophish-ng.git
cd gophish-ng
go build
```

## Running

```bash
./gophish
```

On first run, login credentials are printed to the console:

```
time="..." level=info msg="Please login with the username admin and the password 4304d5255378177d"
```

Open `https://localhost:3333` in your browser.
