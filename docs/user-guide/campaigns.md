# Campaigns

Campaigns in Gophish-NG work the same as upstream Gophish. A campaign ties together:

- A **sending profile** (SMTP server)
- An **email template** (with optional attachments)
- A **landing page**
- A **target group** (list of recipients)

## Creating a Campaign

1. Go to **Campaigns → New Campaign**
2. Fill in the campaign name, email template, landing page, URL, and sending profile
3. Select a target group
4. Set a launch date (or launch immediately)
5. Click **Launch Campaign**

## Monitoring Results

Click the **chart icon** next to a campaign to open the results view. See [Campaign Results](campaign-results.md) for details on the status indicators.
